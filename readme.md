### QuickStart Guide
It is a quick guide to get you up and running. It covers the following topics. Before beginning please complete [Setting Up Your Windows](https://github.com/morrisoncm/documentation/wiki/Setting-Up-Your-Windows). The [repositories](https://github.com/morrisoncm?tab=repositories) below can be viewed and cloned. For a detailed explanation please see the [wiki](https://github.com/morrisoncm/documentation/wiki).
